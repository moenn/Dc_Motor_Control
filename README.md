### 电机

##### 电机引脚
| 电机引脚(从上至下) | 功能                         |
|----------|------------------------------|
| 1脚      | 电机电源负极                 |
| 2脚      | 编码器电源负极               |
| 3脚      | C1编码器A相                  |
| 4脚      | C2编码器B相                  |
| 5脚      | 编码器电源正极（3.3V或者5V） |
| 6脚      | 电机电源正极(12v)            |

##### 电机参数

| 参数 | |
|-----------------|----------------------------|
| 额定电压     | 12v                         |
| 转速       | 6V时为165rpm，12V时为350rpm      |
| 空载转速   | 350rpm                        |
| 效率点          | 2.0kg.cm/285rpm/5.0w/0.65A |
| 功率点          | 5.8kg.cm/180rpm/9.0w/1.65A |
| 堵转参数        | 12.0kg.cm                  |
| 堵死电流        | 5.5A                       |
| 减速器减速比    | 1:34                       |
| 霍尔分辨率      | 11                         |
| 精确减速比 | 341.2(ppr)                      |
